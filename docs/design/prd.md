# 说文 saynow PRD

## 定位

一款运行在 Windows 端的小而美语音输入工具。用户在任意输入框中按住快捷键说话，软件调用在线多模态模型识别，并把结果自动填入当前聚焦输入框。

## 页面

- 首页：使用数据看板和最近语音识别记录。
- 配置：模型 API 供应商配置和启动语音识别快捷键配置。
- 数据：识别记录、自定义词库、自定义风格提示词。
- 反馈：MVP 保留入口，暂不填充业务内容。

## 核心流程

1. 用户配置模型供应商、API Key、模型和快捷键。
2. 软件最小化到托盘后台运行。
3. 用户在任意应用中聚焦输入框。
4. 按住快捷键录音，松开结束。
5. 软件组合音频、自定义词库、风格提示词和相关历史记录。
6. 调用 MiMo-v2.5 或 Qwen3.5-Omni 等 OpenAI-compatible 多模态 API。
7. 自动写入识别文本。
8. 保存记录并更新统计。

## MVP 边界

不做实时字幕、会议转写、云同步、账号系统、插件市场和音频批量导入。
